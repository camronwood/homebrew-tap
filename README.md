# camronwood/homebrew-tap

Homebrew tap for [Neural Junkie](https://github.com/camronwood/neural-junkie).

## Install

```bash
brew tap camronwood/tap
```

### macOS (cask)

```bash
brew install --cask neural-junkie
brew upgrade --cask neural-junkie
brew uninstall --cask --zap neural-junkie   # removes ~/.neural-junkie
```

Installs the GitHub Release `.dmg`. Ollama is bundled on macOS.

### Linux (formula)

Requires [Homebrew on Linux](https://docs.brew.sh/Homebrew-on-Linux).

```bash
brew install neural-junkie
brew upgrade neural-junkie
brew uninstall neural-junkie
```

Unpacks the published `neural-junkie_<version>_amd64.deb`. **Do not** use `--cask` on Linux.

## Maintainer

Cask + formula are regenerated by `scripts/bump-homebrew-tap.sh` in the main repo when a release is published.

```bash
cd /path/to/neural-junkie
./scripts/bump-homebrew-tap.sh v1.2.0-beta.5 ../homebrew-tap
```

Then commit and push this tap repo (`Casks/neural-junkie.rb`, `Formula/neural-junkie.rb`).

See [neural-junkie/docs/HOMEBREW.md](https://github.com/camronwood/neural-junkie/blob/main/docs/HOMEBREW.md).
